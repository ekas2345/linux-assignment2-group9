1. What are special path symbols ., .., and ~ used for in Linux?
. refers to the current directory, .. refers to the parent directory, and ~ represents the home directory of the current user.

2. Briefly describe the purpose of these directories: bin/, etc/, var/, and media/.
bin/ contains essential command binaries, etc/ stores configuration files, var/ holds variable data like logs or caches, and media/ is used for removable media like USB drives.
